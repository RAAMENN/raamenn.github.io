Create your own blog with hexo and Github

Posted on 2017-01-29 | In technical

This is a brief guide to set up your personal blog(on macOS). Using hexo and Github can make your blog available to everyone. Thanks to these amazing developers.

Build your environment

First of all, you should build environment on your Mac to create your own blog.

git

Install git. Follow the instructions. That’s pretty easy.
To check if you’ve successfully installed. Type

in your terminal.If it’s installed correct, it will return it’s current version installed on your Mac.

Node.js

hexo developed on Node.js, install it as well.

hexo

We are almost done! Now install hexo with your terminal.

npm install hexo-cli -g

Type

hexo -v

to see if it’s installed correctly.

hexo

Congrats! We have finished build our environment! Let’s see what hexo can do.
However, you still have something to do before you can use it.
Go the the hexo folder in terminal to initialize hexo first.

npm hexo init

than install some missing package

npm install

Create a local blog

We are almost there!

hexo g
hexo s

You will see the following message:

INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

Open it in your brower. You will surprising find your personal website is there!

Make it your own

Well… It’s still like everyone else’s blog. We should personalize it first.

Modify config.yml

Modify the _config.yml in the hexo folder.
Check hexo offical docs to make configurations.

Theme

Every time you make changes, don’t forgot these things

hexo g
hexo s

Until now you can see the changes you made.

Personalize with theme

I know, I know, you hate the default theme. hexo know it as well. That why you have so many theme to use. Agian thanks to those who made these themes.

Github

If the blog is for your own, you should skip this part. I gonna explain how to make it online in this part

Create an account

Create an account on Github first. It should be easy.

New repository

After created an account, you can use the new repository now. You should name the new repository: xxx.github.io

Make it online

edit the git config:

git config --global user.name "yourname"
git config --global user.email "youremail"

edit your _config.yml

deploy:
    type:git
    repository: git@github.com:username/xxx.github.io.git
    branch:master

SSH

click here to learn how to connect to GitHub with SSH.

Press it

hexo g
hexo d

Congrats!!Now you can visit your own blog on xxx.github.io!

Your first blog

Your blog is still empty. Let’s post our first blog!

hexo n "myFirstBlog"

hexo will automaticlly create a myFirstBlog.md file for you. You can now edit your first blog.
Don’t forget when you make changes!

hexo g
hexo d

Visit xxx.github.io to see your first blog.

Macdown

Macdown is a open source software used to edit .md file on Mac.
I highly recommend it.

Markdown syntax

Find problems when editing a markdown? Here is the tutorial for markdown syntax.

Thanks

I hope this brief tutorial helped you. Have fun with your own blog!
